# Truth Table Generator

A desktop application built with **Java** and **Java Swing** that generates truth tables for propositional logic expressions. The application allows users to enter a logical expression, automatically identifies the variables involved, evaluates the expression for every possible combination of truth values, and displays the complete truth table in an easy-to-read interface.

## Features

* Generate truth tables for logical expressions
* Automatically detect variables used in the expression
* Evaluate expressions for every possible truth value combination
* User-friendly graphical interface built with Java Swing
* Fast and lightweight desktop application
* Input validation for logical expressions

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)

## How to Run

1. Clone the repository.
2. Open the project in your preferred Java IDE (IntelliJ IDEA, Eclipse, or NetBeans).
3. Build and run the project.
4. Enter a logical expression and generate its truth table.

## Example Expressions

* `A & B`
* `A | B`
* `!A`
* `(A & B) -> C`
* `(A | B) & !C`
